# PreQursor Frontend

Frontend application for **PreQursor**, an esports platform where players can join competitive matches and tournaments.

This application provides the user interface for players to discover matches, join competitions, and manage their participation.

---

## Features

- User authentication (login & signup)
- Browse available matches
- Join esports matches
- Wallet balance display
- Enrolled matches tracking
- Participated matches history
- Screenshot upload for match verification
- Responsive UI for desktop and mobile

---

## Tech Stack

- React
- Typescript
- Tailwind CSS
- Axios
- React Router

---

## Setup

Clone the repository:

```bash
git clone https://github.com/Abdul-Wasay-008/PreQursor-FE.git
cd PreQursor-FE
```
Install dependencies:
```bash
npm install
```
Run the development server:
```bash
npm start
```
